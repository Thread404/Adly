# Adly
Adly.ai is an AI tool that generates advertisement posters and captions for the poster using only text inputs from the user describing the poster. This app has the following features:
- Text-to-image generation
- Text-to-text generation
- Text editor tools
- image uploads and downloads
For future updates, the application will have the following features:
- Basic shapes drawing tools
- Free-form drawing tools
- Image filtering
- Image segmentation
Adly.ai uses diffusion models to generate images from text and from that text, it also uses transformers to generate text captions (with proper prompt engineering) for the image based on the given image description.
